# Laser Raster/Vector Engraving

- Parameters: resolution, beam diameter, step-over, overscan, feed, power map, dithering, invert.
- PWM mapping: normalize power to controller S-range (GRBL/Smoothie/Mach3); optional gamma correction.
- Scanline generation: bidirectional with overscan; M4 dynamic power where available.
- Vector hatch/outline: hatch angle, spacing, joins.
- Safety notes for FR-4 and painted mask approaches.
